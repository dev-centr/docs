= Constituent Tools

Welcome to the tools section. Here you will find documentation for the various tools and repositories that make up the Dev Center ecosystem.

Some tools are documented in their own repositories (e.g. MSI Generator); others are documented here (e.g. dprint, dlangui).

== Original Contributions
These tools are original works developed as part of this ecosystem.

* xref:tools/msi-installer-generator.adoc[MSI Generator]

== Maintained Forks
These tools are maintained forks of upstream projects, often incorporating local custom patches to improve the developer experience. We intend to contribute the features introduced here to the upstream projects, where possible. `dprint` is the only exception, as they have been jerks.
